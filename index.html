A neat way to create a Professor list
August 14, 2017 / by Enes Dilber

As an udergrad student who searches us colleges for PhD i tried to find Professors who work on the topics that i'm interested in. At first i've created a spreadsheet for Professors, but when the number of rows increased, i was confused by the information that the sheet represents. I mean we are statistician. When the data size increase, we need to extract a more compact imformation from data with visuals, models etc. So i decided to write a code that maps the professor information. In this example, the map marker changes with the qs score of the institue. And when users click the marker, they see the professors in that institue.
