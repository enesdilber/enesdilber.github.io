Resources for Machine Learning

I am also a newbie in this area, but once you want to learn new things or write your code a little fancier, there are many free amazing sources. Let me introduce you the ones that i used and the ones that i am still using a lot.


Online Courses


1   Machine Learning by Stanford University

Well, this is by far the most popular online course for machine learning. They prepare the course very well and Andrew Ng is so good at delivering it. They use MATLAB (and Octave) for implementation. Overall, it's a good start to ML.

2   Applied Machine Learning by Microsoft

This is a "Swiss army knife course for data analysis." It has 4 modules for different type of data. Namely; time series, spatial, image, text. The course is a nice introductary course for different areas of data analysis. By the way, they use R for Spatial and Time series data and python for Image and Text data.

3   Neural Networks by Hugo Larochelle

Altough the coursera course of Geoffrey Hinton is the most popular Neural Network course, i like the materials and videos of Hugo Larochelle more.

4   Convolutional Neural Networks for Visual Recognition by Stanford University

A must course for people who use Deep Learning to classify images. Obviously the good guy Andrej Karpathy put a lot to create this magnificant course. There is an introductary part where they cover Machine Learning, then course continues with CNN which is the state of the art of object recognition techniques.

Books


1   Advanced R by Hadley Wickham

Without a doubt, Hadley Wickham is one of the most important contributor to R Language. In this book he provides a deep understanding of R. For me, the best part of the book is functional programming part. As a statistician, i am not familier with different paradigms of languages like functional, object-orianted and procedural. Since at the heart, R is a functional programming language. This book helps you to understand what a functional programming language, and how should you write code in a "functional way".

2   Deep Learning by Ian Goodfellow, Yoshua Bengio and Aaron Courville

Currently, this book is the only book that contains up to date information about Deep Learning. It gives a formal background for the concept. A must resource for Deep Learning practioners and researchers.
