get_d2 = function(X, x_bar, S){
 
  n = dim(X)[1]
  p = dim(X)[2]
  x_bar = matrix(x_bar, p, 1)
  Sinv = solve(S)
  
  d2 = numeric(n)
  
  for(i in 1:n){
    d2[i] = t(X[i,] - x_bar) %*% Sinv %*% (X[i,] - x_bar)
  }
  
  return(sort(d2))
}

6. Inference to $\mu$ (Hotelling's $T^2$)¶

$$ H_0 : \mu = \mu_0 \\ \\ H_1 : \mu \neq \mu_0 $$
$$ T^2 = n(\bar{X} - M_0)^T S^{-1} (\bar{X} - M_0) \\ $$$$ reject \ H_0 \ if \ T^2 > c \ where; \\ $$

$$c = \frac{(n-1) \ p }{n-p}F_{p,n-p,\alpha} \\ $$

Because:

$$ T^2 \sim \frac{(n-1) \ p }{n-p}F_{p,n-p}$$

Lets test above sample with: $$ H_0 : \mu = \left[ {\begin{array}{c} 0\\ 0\\ 0\\ 0\\ \end{array} } \right]\\ \\ H_1 : \mu \neq \left[ {\begin{array}{c} 0\\ 0\\ 0\\ 0\\ \end{array} } \right] $$

and $\alpha = 0.05 $
